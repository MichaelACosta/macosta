Master in computing at UFPel and Front-end developer at CI&T, BR.

I like to write code to optimize the performance of parallel systems and create web applications in my spare time.

In my master's work was to propose and implement a scheduler for transactional memories in software (STM) called Lups Transactional Memory Scheduler (LTMS), in this work I raised and validated the hypothesis that a thread scheduler that obtains context about the execution of the application and of the hardware architecture used can reduce the number of aborts and the execution time of applications that use STMs.
